Abstract

Collision-free motion is still not enough when localization can drift into failure.

Long-range UAV navigation in GNSS-denied feature-sparse outdoor environments is hard because LiDAR-inertial odometry can become ill-conditioned even when a geometric path exists. Once the estimator weakens, drift accumulates and the mission can collapse before the vehicle reaches the goal.

AUG addresses this with a scalable perception-aware planning framework that couples long-horizon connectivity reasoning with estimator-aware online replanning. It incrementally builds a lightweight free-space topological graph, generates local MINCO trajectories on a robocentric ROG-map, and explicitly optimizes yaw so the sensor keeps pointing toward informative structure.

Contributions

Three additions make the planner work at long range without a global ESDF.

01

Topology-guided global routing

Builds a free-space graph from bubble-based decomposition and feature-aware node selection, then searches corridors that stay estimation-friendly as well as collision-free.

02

Robocentric local planning

Uses a sliding ROG-map and MINCO optimization to keep runtime and memory nearly constant while still generating smooth, real-time position trajectories.

03

Perception-constrained yaw

Searches yaw profiles that keep nearby informative structures visible to LiDAR, reducing estimator degeneration in sparse scenes where geometric heuristics are brittle.

Method

Global route, local trajectory, and yaw are optimized as one planning stack.

AUG first forms incremental topological connectivity from local free-space bubbles and feature density, then selects a global route that avoids perceptual dead zones. The chosen route feeds a lightweight local optimizer that only operates on a sliding map window.

In parallel, a yaw planner evaluates where the LiDAR can retain useful features and refines heading so the vehicle sees more stable constraints during the flight. This is the main difference from purely geometric yaw alignment.

A

Incremental topology graph

Nodes are created from connected free-space components and filtered by feature support, allowing global routing without a large ESDF volume.

B

ROG-map trajectory optimization

A robocentric map representation supports smooth local planning with bounded memory, which is critical for kilometer-scale missions.

C

Yaw search and refinement

The planner explicitly scores sensor orientation by expected feature quality, then refines a smooth yaw profile within dynamic limits.
