Title: AUG: Perception-Aware UAV Graph-Based Planning for GNSS-Denied Navigation in Feature-Sparse Environments

1. General description
This video demonstrates AUG, a scalable perception-aware planning framework for long-range UAV navigation in GNSS-denied, feature-sparse outdoor environments. The method combines an incremental free-space topological graph for global routing, a robocentric ROG-map with MINCO-based local trajectory optimization, and a perception-constrained yaw planner that steers the LiDAR toward informative structures to improve LiDAR-inertial odometry conditioning. The video shows simulation and real-world flight results, highlighting safer routes, lower drift, and robust onboard execution.
